    实验步骤

    1. 编写池化层IP核

    • 参照所给出的卷积IP核的HLS工程，编写池化加速的IP核代码。需要注意的是，池化IP核需要根据mode来选择池化计算方式。可定义接口如图1-26所示。

    void Pool(ap_uint<16> CHin, ap_uint<16> Hin, ap_uint<16> Win,
           ap_uint<8> Kx, ap_uint<8> Ky, ap_uint<2> mode,
           Dtype_f feature_in[], Dtype_f feature_out[])
    图1-26 池化层IP核接口定义

    • 代码编写完成后，参考实验1指导书2.3.1节“利用HLS生成IP核”，对所设计的池化IP核进行仿真，确认代码功能正确后，点击工具栏的按钮进行综合，然后是C/RTL联合仿真，最后点击工具栏的按钮生成池化IP核。

    2. 更新Block Design

    • 实验包中的base目录提供了已经构建好Block Design电路图的Vivado模板工程。模板工程Block Design中的池化IP核是一具无功能的空壳。同学们需要将自行设计的IP核更新到模板工程的Block Design电路图中，并生成比特流。

    • 更新时，先将自行设计的池化IP核添加到Vivado的IP库中，如图1-27所示。

    图1-27 添加IP核至Vivado中

    • 双击打开base.bd，Vivado将提示需要更新IP核。此时，按照提示点击“Report IP Status”即可，如图1-28所示。

    图1-28 打开Block Design并按提示点击“Report IP Status”

    • 在随后弹出的IP Status窗口中，点击“Upgrade Selected”按钮，如图1-29所示。

    图1-29 更新池化IP核

    • 随后将弹出更新完毕的提示对话框和“Generate Output Products”的对话框，分别点击“OK”按钮和“Generate”按钮即可。

    • 更新完毕后，点击“IP Status”窗口中的“Rerun”按钮，确认IP核是否更新成功，如图1-30所示。

    图1-30 确认IP核是否更新成功

    3. 生成并导出Overlay

    • 点击Vivado工具栏中的按钮以生成比特流。生成完毕后，导出Block Design所对应的.tcl文件如图1-31所示。

    图1-31 导出Overlay中的.tcl文件

    • 将/xx/base/base.tcl文件和/xx/base/base.runs/impl_1/base_wrapper.bit文件拷贝到实验包中的mnist目录，并将比特流文件重命名为base.bit。

    • 接下来，在文件资源管理器的地址栏输入\\192.168.2.99\xilinx，以xilinx作为用户名和密码登录，即可在Windows系统中访问PYNQ-Z2的贡献文件夹，如图1-32所示。

    图1-32 访问共享文件夹

    • 将实验包的mnist目录整个拷贝至\\192.168.2.99\xilinx\jupyter_notebooks目录内。

    4. 网络推理测试

    • 接下来，需要打开PYNQ-Z2的嵌入式Linux终端，以进行CNN的推理测试。

    • Linux终端的打开方式有两种：

    • 第1种方法是在Chrome浏览器中访问192.168.2.99:9090，以xilinx作为密码登录。然后点击New按钮创建终端，如图1-33所示。

    图1-33 创建新的终端

    • 第2种方法则比较简单，只需用Secure CRT工具连接PYNQ-Z2的串口即可，如图1-34所示。

    图1-34 在串口调试助手中使用嵌入式Linux终端

    • 在终端中输入以下命令：

      # cd ~/jupyter_notebooks/mnist

      # python3 mnist_inference.py

    • 稍等片刻后，程序将完成Overlay加载及其他初始化操作，如图1-35所示。

    图1-35 初始化完成

    • 在文件资源管理器中访问\192.168.2.99\xilinx\jupyter_notebooks\mnist\data，使用画图软件打开1.jpg图片，如图1-36所示。

    图1-36 打开并绘制数字

    • 在画图工具中画出任意数字，保存1.jpg文件。随后在Linux终端中输入回车键。此时，测试程序将对刚刚绘制的数字进行识别，如图1-37所示。

    图1-37 利用CNN推理识别1.jpg中的数字

    • 同学们可绘制其他数字以测试CNN的推理功能。
